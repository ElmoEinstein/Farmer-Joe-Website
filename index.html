Farmer Joe
Farmer Joe is a extended version off a game I made for a Game Jam. I worked on it for around 9 months but probably won't continue as I lost passion for the project and some features are buggy because I was new to Unity upon creating the game.

Game Jam Version

Farmer Joe was originally created for the Ludum Dare 52 game jam as a project with my friend. It is a game about a farmer trying to make as much money as possible in a short amount off time. The game did pretty well so I decided to keep on working on it but sadly my friend did not share my passion.

Game Jams
Game Jams are game making competitions held online for game developers to test their skills in a time limit (usually 2-3 days). These competitions have been a massive part of my game development and multimedia journey, growing my skills substantially over a very short amount of time.

Ludum Dare 51

I mostly participate in a specific Game Jam called Ludum Dare, which is a 3 day competition and one of the most popular Game Jams with roughly 2000 participants each competition. The first game I made for Ludum Dare was called A Messed Up Elephant And Some Mean Racoons. It wasn't very good, had many bugs, was glitchy and overall wasn't fun but nevertheless, I was still pround of it.

Ludum Dare 52

The next Game Jam I participated in, near the end of Year 9, was a Ludum Dare, in which me and a friend created a early version of Farmer Joe. The game was about a farmer trying to make as much money as possible in a set amount off time. I did all the programming and he did most of the art, we ended up placing in the top 15% of games.

Mini Jam 130

After that, I decided to undertake my first none Ludum Dare game jam in the form of a Mini Jam, which is a 3 day Game Jam that occurs every 2 weeks. I worked with another random game developer (in which I never spoke too) and I did the programming and some art, while he did majority off the art. The game was called Lunar Artillery and it was about destroying waves off aliens. While the game didn't turn out the best in its presentation, I was proud off myself for creating a fully functioning wave system and gun system in such a short amount off time.

Ludum Dare 54

The next Game Jam game I made solo, at the very start off year 10 was one that I was very pround off. It was an ant cononley simulator call Fun Ant Game, where you had to work on expanding and building a cononley with different resources, while strategically placing your buildings to make the most off your resources. I managed to code a lot off systems in the short time limit, create my first ever music score and a lot off people ended up spending half an hour or more playing my game.

Boss Russ Game Jam

Boss Rush Game Jam is a very long 30 day Game Jam where you have to create a boss rush based on a them. Sadly I only got a week into because off sudden commitments that appeared later on, but I was still very happy with some off the art and concepts I made. My concept was that I was going to create a game about a dog with a katana (called Katana Kenji), fighting off enemies. I made a few level with concept art and most off the first boss (gameplay on my Twitter) and was really pround off all the art and animations I made.

About me
My name is Kasper and I am a Australian Game Developer born in 2008. My hobbies include activities such as Rock Climbing, going to the gym and of course Game Development.

I started learning Game Development in 2021, right after COVID as I had always been interested in the industry. I worked on many different games and participated in lots of game jams online.

Nearing early 2024, I decided I wanted to expand my knowledge on the multimedia industry and worked on growing my skillset. Whilst I already knew how to create pixel art and code in Unity and GameMaker, I still didn't have any knowledge in creating 3D models, music, photography and creating websites.

In early 2024, I learnt html and css near the middle of the year learnt Java Script using 3JS. Mid way through 2024 I also decided to take creating music more seriously, making songs in a synthwave style for my games and school projects. I also picked up the basics of Blender to create the portfolio you are reading right now!
